### Task 2: Hook `useProfile` + `useReferral`

**Files:** Create `features/profile/use-profile.ts`, `features/profile/use-referral.ts` + test ciascuno.

**Interfaces:**
- `useProfile(): { profile: { identifier: string; name: string | null; plan: string; planExpiresAt: string | null } | null; loading: boolean; error: string | null; loggedIn: boolean; logout: () => Promise<void> }`. Fetch `GET /api/auth` (credentials include); 401 → `profile=null, loggedIn=false` (non è "error"). `logout()` = `POST /api/auth {action:"logout"}` poi `profile=null`.
- `useReferral(): { code: string | null; signups: number; paid: number; loading: boolean; error: string | null; claim: (code: string) => Promise<{ ok: boolean; error?: string }> }`. Fetch `GET /api/referral/stats`; **403 → `code=null`** (nessun codice, NON errore); 200 → `{code,signups,paid}`. `claim(code)` = `POST /api/referral/claim {code}` → su ok ri-fetcha stats.

- [ ] **Step 1-5 (per hook):** test con fetch mockato (200 loaded+mapped; 401/403 → logged-out/no-code, non error; logout/claim chiamano il POST giusto). RED → implementa (alive guard, credentials) → GREEN. Commit: `feat(sp5): useProfile + useReferral hooks`
